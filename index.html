The Box Model

The CSS box model as a whole applies to block boxes and defines how the different parts of a box — margin, border, padding, and content — work together to create a box that you can see on a page.

Parts Of A Box Description Sizing Property
Content Box The area where your content is displayed.
  • inline-size
  • block-size
  • width
  • height
Padding Box The padding sits around the content as white space.
  • padding
  • related properties
Border Box The border box wraps the content and any padding.
  • border
  • related properties
Margin box The margin is the outermost layer, wrapping the content, padding, and border as whitespace between this box and other elements.
  • margin
  • related properties

Syntaxes, For Writing Box Model CSS Code

Inline CSS Internal Style Sheets Ruleset
  • It’s possible to write CSS code directly within HTML code using inline styles.
  • This code <p style="color: red">I'm learning to code!</p> renders:

  • I'm learning to code!

  • The style attribute is addded within the HTML element's opening tag
  • It’s important to know that inline styles are a quick way of directly styling an HTML element, but are rarely used when creating websites.
  • CSS code can be written in an HTML file by enclosing the code in <style> tags. Code surrounded by <style> tags will be interpreted as CSS syntax.
  • <head> <style> p { color: red; font-size: 20px; } </style> </head>
  • Note how the syntax of the CSS code matches (for the most part) the syntax you used for inline styling. The main difference is that you can specify which elements to apply the styling.
  • An internal stylesheet has certain benefits and use cases over inlines styles, but once again, it’s not best practice
  • Developers avoid mixing code by storing HTML and CSS code in separate files (HTML files contain only HTML code, and CSS files contain only CSS code).
  • You can create an external stylesheet by using the .css file name extension, like so: style.css
  • You can use the <link /> element to link HTML and CSS files together. The <link /> element must be placed within the head of the HTML file. It is a self-closing tag and requires the following attributes:
    1. href — like the anchor element, the value of this attribute must be the address, or path, to the CSS file.
    2. rel — this attribute describes the relationship between the HTML file and the CSS file. Because you are linking to a stylesheet, the value should be set to stylesheet.
  • A URL path would look like this <link href="https://www.codecademy.com/stylesheets/style.css" rel="stylesheet" />
  • If your CSS and HTML file are in the same directory or local file you can specify a relative path like this: <link href="./style.css" rel="stylesheet" />
